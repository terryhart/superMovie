# 🔥基于MVP+RxJava+Retrofit的电影资源聚合APP



手机端更舒适地下载高清电影，热门电影，美剧，每日更新。（效果图更新不及时，以最新app实际效果为准）

## 1.4.0版本已发布，新增投屏助手，本地资源，在线资源皆可投屏到电视、投影。
## 欢迎下载体验 ：https://fir.im/btmovie
